# PDF Craft 云

将您的 PDF 文档转换成更易于阅读、编辑和分享的格式。PDF Craft 云自动完成转换工作，让您可以专注于按自己的方式使用内容。

## 您可以做什么？

### 将 PDF 转换为电子书
将任何 PDF 文档转换为 EPUB 文件，这样您就可以在 Kindle、iPad 或任何电子阅读器上阅读。非常适合在旅途中阅读研究论文、报告或书籍，享受更好的文本自动调整和字体大小控制。

### 将 PDF 转换为可编辑文本
将 PDF 文档转换为 Markdown 格式，使您可以轻松编辑内容、提取信息或将文本用于自己的项目。非常适合从文章中记笔记、为演示文稿改编内容或创建文档。

## 如何使用

PDF Craft 云提供了一个简单的工具，叫做 **PDF 转换器**，它能完成所有复杂的工作：

1. **指定您的 PDF**：告诉它您的 PDF 在哪里 - 可以提供网页链接，也可以从您的电脑上传文件
2. **选择格式**：选择您想要 EPUB 电子书还是 Markdown 文本文件
3. **让它运行**：转换器自动处理一切 - 它会处理您的 PDF 并返回转换后的文件

就是这样！无需担心复杂的步骤或设置。

## 智能功能

转换器包含在后台自动运行的智能功能：

- **进度监控**：实时查看转换进度
- **错误处理**：如果 PDF 的某些部分难以读取（如扫描页面或特殊格式），转换器可以跳过这些部分并继续处理其余内容
- **自动处理**：只需提供您的 PDF 和所需格式 - 其他一切都会为您处理

## 常见使用方式

### 随时随地阅读
- 将冗长的 PDF 报告转换为 EPUB 格式
- 传输到您的电子阅读器，随时随地舒适阅读
- 享受比标准 PDF 更好的文本自动调整和字体大小控制

### 内容编辑和再利用
- 将 PDF 文章转换为可编辑的 Markdown 文本
- 为您自己的文档提取和修改内容
- 提取引用或段落而无需重新输入

### 文档处理工作流程
- 您可以考虑将其与文档整理工具一起使用，自动转换和归档 PDF
- 如果您需要处理 PDF 内容，通常与文本分析工具配合使用效果很好
- 可以与笔记系统结合使用，从 PDF 来源中捕获信息
- 与云存储服务配合使用效果很好，可以自动保存转换后的文件

## 实际应用示例

**研究人员**：将学术论文从 PDF 转换为 Markdown，便于记笔记和管理引用。

**商务人士**：将季度报告转换为 EPUB 格式，在平板电脑上通勤时阅读。

**团队管理者**：设置自动化工作流程，转换传入的 PDF 文档并将其发送给合适的团队成员。

## 开始使用

只需将"PDF 转换器"模块拖入您的工作区，并将其与您的 PDF 来源连接。该工具会处理从此开始的所有事情，将转换后的文件直接送到您需要的地方。

无需技术知识 - 只要您能提供 PDF 位置并选择格式，就可以开始使用了。
